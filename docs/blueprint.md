# **App Name**: Automate Pro

## Core Features:

- Hero Section: Full-screen introduction with name, title, animated background, and clear call to action buttons for project viewing and contacting.
- About Me Section: Professional summary of experience, skills, and focus with optional profile image and smooth fade-in animation.
- Skills Showcase: Interactive display of Python, Django, Flask, and other skills, with hover glow effects.
- Projects Portfolio: Project cards highlighting Log Management System, Inventory Management System, and other projects, including tech stack badges and buttons for GitHub and live demos.
- Experience & Tools Timeline: An elegant display of past experience and the tools utilized within those roles using a timeline or grid layout to highlight the most relevant aspects of the career.
- Contact Form: Minimal contact form for direct inquiries including name, email, and message fields, accented with a gold submit button.
- Navigation: Sticky left sidebar with icon-based navigation that highlights active sections in gold. It includes tooltips, smooth scrolling, and is collapsible on mobile, replaced by a top hamburger menu.

## Style Guidelines:

- Primary color: Gold (#d4af37) for a premium and professional accent.
- Background color: Dark gray (#0a0a0a) to provide contrast.
- Text Primary color: White (#f5f5f5) to ensure high legibility.
- Secondary accent color: a slightly darker shade of gold (#b8962e), used specifically on hover states to convey feedback about user interactions.
- Body and headline font: 'Inter', a sans-serif font with a neutral and modern aesthetic for clean readability.
- Utilize Lucide or Heroicons for minimal and clean icons.
- Single-page vertical layout with CSS scroll-snap for a premium feel.
- Subtle and smooth entrance animations using Framer Motion and CSS scroll-snap, inspired by Vercel, Linear, and Apple-style UIs.
